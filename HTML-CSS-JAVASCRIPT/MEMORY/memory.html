MEMORY 26-11-2020

Introduction

This memory was designed between 12 and 26 November of 2020. Click the image to start game.

The focus is on the 17 goals of AGENDA 2020. Every card is a point of Agenda, match it with the same card in the less attempts

The game on start ask you the difficulty, select the quantity of cards you want between 2 and 17. Then you have to click on two cards, they will be showned andm if they are the same they will disappear; if not they will turn closed again. The game ends when all the cards are desappeared.

WHAT IS AGENDA 2030

This Agenda is a plan of action for people, planet and prosperity signed in 2015 by 193 nations members of ONU

The plan provides for helping nation above the 17 points to improve the human prosperity on earth, take care of planet, stop pollution, stop wars and help poor peolple in all the world.
